boolean

Dating web app for computer science students built in JavaScript and using bootstrap

Images

Description

One of my first projects in my first year at Champlain, boolean is a very simple dating application that matches you by percentage to another person according to your answer to 8 true/false question. While I'm not convinced I would trust it to find the love of my life, it served its purpose well by allowing me to play around with javascript, bootstrap and css while fulfilling strict requirements.

Challenges and Features

  • Fully responsive as part of the learning experience
  • Integrates many UI libraries
  • First basic experience with forms
